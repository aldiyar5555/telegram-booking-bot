# Telegram Booking Bot 🤖

Интерактивный бот-визитка для записи на услуги и демонстрации портфолио.

### ✨ Основные функции:
* **Интерактивное меню**: использование Inline-кнопок для навигации.
* **Динамическое обновление**: редактирование сообщений без лишнего спама в чате.
* **Связь с разработчиком**: быстрые ссылки на GitHub, FL.ru и личные сообщения.
* **Безопасность**: использование `.env` для защиты API токенов.

### 🛠 Стек технологий:
* Python 3.11+
* Aiogram 2.x
* Python-dotenv

### 🚀 Запуск:
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` и добавьте свой `BOT_TOKEN`.
3. Запустите бота: `python main.py`
